# Function to detect outliers using IQR (Interquartile Range) method
def detect_outliers(df):
outliers = {}

# Numerical columns for outlier detection
numerical_columns = [
'Age', 'Sleep Duration', 'Quality of Sleep',
'Physical Activity Level', 'Stress Level',
'Heart Rate', 'Daily Steps'
]

for col in numerical_columns:
Q1 = df[col].quantile(0.25)
Q3 = df[col].quantile(0.75)
IQR = Q3 - Q1
lower_bound = Q1 - 1.5 * IQR
upper_bound = Q3 + 1.5 * IQR
outliers[col] = df[(df[col] < lower_bound) | (df[col] > upper_bound)]

return outliers

# Function to calculate summary statistics
def summary_stats(df):
summary = {}

summary['Average Age'] = df['Age'].mean()
summary['Average Sleep Duration'] = df['Sleep Duration'].mean()
summary['Average Quality of Sleep'] = df['Quality of Sleep'].mean()
summary['Average Physical Activity'] = df['Physical Activity Level'].mean()
summary['Average Stress Level'] = df['Stress Level'].mean()
summary['Average Heart Rate'] = df['Heart Rate'].mean()
summary['Average Daily Steps'] = df['Daily Steps'].mean()

summary['Gender Distribution'] = df['Gender'].value_counts()
summary['BMI Category Distribution'] = df['BMI Category'].value_counts()
summary['Sleep Disorder Distribution'] = df['Sleep Disorder'].value_counts()

return summary

First Dataset:

The first dataset, with 2392 records and 15 variables, is of high quality. There are no missing values, which ensures the integrity of the analyses. However, the GPA values seem inconsistent with the expected scale (2 to 4), so a review of these values is necessary. The dataset is diverse, including variables such as study time, absences, and extracurricular activities, which allows for a thorough analysis of factors influencing GPA. The detection of outliers is limited, indicating overall consistency in the data.

Second Dataset:

The second dataset, with 374 records and 13 variables, is also of high quality. It is free from missing values, and outlier analysis did not reveal significant anomalies in key variables such as sleep duration and stress levels. The variables provide a comprehensive view of lifestyle habits and health aspects, facilitating the assessment of relationships between these factors and sleep quality. Although the dataset is smaller compared to the first one, it remains sufficiently varied for meaningful analysis.

Warning :

Although both datasets are of high quality, they focus on very different subjects, which is evident from the average age of the individuals represented. The first dataset pertains exclusively to students, while the second concerns adults. Consequently, when correlating these datasets, it is crucial to approach the analysis with a critical perspective, taking these demographic differences into account.
