Modeling Assumptions1. Linear Regression assumes a linear relationship between the features and the target variable, which worked well for the Student Performance dataset but struggled with the Sleep Health dataset, where the relationships might be more complex.
2. Ridge and Lasso Regression are regularized versions of linear regression. Ridge penalizes large coefficients, and Lasso can shrink coefficients to zero, effectively performing feature selection. In these cases, Ridge performed comparably to Linear Regression, while Lasso underperformed due to excessive regularization.
3. Gradient Boosting and XGBoost assume that the relationships can be captured through an ensemble of weak learners (decision trees), which explains why these models performed best on the Sleep Health dataset, which likely contains complex, non-linear relationships.
4. Random Forest is an ensemble model that tends to reduce variance through averaging multiple decision trees, but it can still overfit, especially if the trees are too deep.
5. Support Vector Regression is sensitive to the kernel and hyperparameters, which could explain why it didn’t perform as well in these comparisons.
6. Decision Trees suffer from high variance and tend to overfit unless pruned, leading to poor generalization as seen in these results.
